# Caderno Temático: Aprendendo com o NotebookLM

## Contexto

Este projeto foi desenvolvido como parte do desafio da DIO com o objetivo de explorar o NotebookLM como ferramenta de aprendizagem ativa baseada em Inteligência Artificial.

O tema escolhido foi o uso do NotebookLM para organização de estudos, criação de resumos, geração de perguntas e construção de conhecimento a partir de múltiplas fontes.

---

# Objetivos de Estudo

- Compreender o funcionamento do NotebookLM.
- Aprender técnicas de engenharia de prompts.
- Utilizar IA para resumir conteúdos complexos.
- Desenvolver um método de estudo mais eficiente.
- Criar um guia reutilizável para futuras pesquisas.

# Engenharia de Prompts

## Prompt 1

> "Resuma os principais conceitos presentes nas fontes em até 10 tópicos."

### Resultado

Foi gerado um resumo geral do conteúdo com os conceitos centrais.

### Dificuldade Encontrada

As respostas ficaram muito amplas.

---

## Prompt 2

> "Explique o conteúdo como se eu fosse um iniciante."

### Resultado

A IA produziu uma explicação mais simples e didática.

### Dificuldade Encontrada

Alguns conceitos técnicos ainda precisaram de exemplos adicionais.

---

## Prompt 3

> "Crie 10 perguntas para revisar este conteúdo."

### Resultado

Foram geradas questões úteis para revisão e fixação.

### Dificuldade Encontrada

# Troubleshooting

### Problema

Respostas muito genéricas.

### Solução

Adicionar contexto e especificar o nível de profundidade desejado.

---

### Problema

Resumo excessivamente longo.

### Solução

Definir limite de palavras ou quantidade de tópicos.

# Miniguia de Estudo

## Resumo Estruturado

### O que é NotebookLM?

NotebookLM é uma ferramenta baseada em IA desenvolvida pelo Google que permite analisar documentos, criar resumos, responder perguntas e auxiliar no processo de aprendizagem.

### Principais Benefícios

- Organização de conhecimento.
- Resumos automáticos.
- Geração de perguntas.
- Apoio à pesquisa.
- Aprendizagem personalizada.

### Boas Práticas

- Utilizar fontes confiáveis.
- Criar prompts específicos.
- Comparar respostas.
- Validar informações importantes.

---

# Glossário

## IA Generativa

Tecnologia capaz de criar textos, imagens e outros conteúdos.

## Prompt

Instrução enviada para a IA.

## Engenharia de Prompt

Processo de criar instruções eficientes para obter melhores respostas.

## NotebookLM

Ferramenta do Google para estudo e pesquisa baseada em IA.

## Aprendizagem Ativa

Método em que o estudante participa ativamente da construção do conhecimento.
